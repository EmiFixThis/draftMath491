\newglossaryitem{ID-based encryption}{
    name={ID-based Encryption},
    description={In an ID-based PKC encryption scheme the user \emph{is} some unique peice of information about the user; i.e., the users name, IP, email address, etc.
    ID-based encryption was proposed by Shamir in 1984, the IBE problem was solved in 2001 with the Boneh-Franklin pairing scheme, and Cock's quadratic residue scheme.}
}
